### ဖောဋ္ဌဗ္ဗာရုံလိုင်း - မနောဒွါရဝီထိ - ဒုတိယဇော - ဝေဒနာက္ခန္ဓာ

**ပစ္စုပ္ပန်အကြောင်းတရား (၄) ပါး**

၁။ ဝတ္ထု (= ဟဒယ = ၆ = ၅၄)က အကြောင်းတရား၊ ဒုတိယဇော ဝေဒနာက္ခန္ဓာက အကျိုးတရား။

၂။ အာရုံ (= ဖောဋ္ဌဗ္ဗာရုံ)က အကြောင်းတရား၊ ဒုတိယဇော ဝေဒနာက္ခန္ဓာက အကျိုးတရား။

၃။ ဖဿ = (က) ကာယသမ္ဖဿ (= ၈)က အကြောင်းတရား၊ ဒုတိယဇော ဝေဒနာက္ခန္ဓာက အကျိုးတရား။

( ခ ) ဘဝင် မနောသမ္ဖဿ (= ၃၄)က အကြောင်းတရား၊ ဒုတိယဇော ဝေဒနာက္ခန္ဓာက
အကျိုးတရား။

( ဂ ) ပထမဇော မနောသမ္ဖဿ (= ၃၄)က အကြောင်းတရား၊ ဒုတိယဇော ဝေဒနာက္ခန္ဓာက
အကျိုးတရား။

(ဃ) ဒုတိယဇောမနောသမ္ဖဿ (= ၃၃)က အကြောင်းတရား၊ ဒုတိယဇော ဝေဒနာက္ခန္ဓာက
အကျိုးတရား။

၄။ ယောနိသောမနသိကာရ (= မနောဒွါရာဝဇ္ဇန်း = ၁၂)က အကြောင်းတရား၊ ဒုတိယဇော ဝေဒနာက္ခန္ဓာက
အကျိုးတရား။
